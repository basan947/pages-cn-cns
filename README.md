# pages-cn-cns

本仓库用于归档和发布多个独立 HTML 页面。这些页面内容不针对任何特定域名或网站，仅作为页面资源存储与分发用途。

## 项目简介

`pages-cn-cns` 是一个面向通用 HTML 页面的归档仓库。仓库内的页面均为独立文件，不依赖外部数据库或后端服务，可直接通过浏览器访问。所有页面内容保持静态，便于长期保存与快速部署。

## 目录说明

```
pages-cn-cns/
├── index.html         # 仓库入口页面（可选）
├── page-1.html        # 示例归档页面
├── page-2.html        # 示例归档页面
└── assets/            # 共用资源文件夹（样式、脚本、图片等）
    ├── style.css
    └── script.js
```

- 每个 `.html` 文件对应一个独立页面。
- `assets/` 目录存放页面共用的静态资源文件。
- 如需新增页面，请遵循现有命名规范（如 `page-xxx.html`）。

## 页面归档说明

归档的 HTML 页面需满足以下条件：

- 页面内容完整，无外部链接依赖（除公用资源外）。
- 文件编码为 UTF-8，确保多语言兼容。
- 页面内不应包含任何指向具体域名或网站的链接。
- 建议在 `<head>` 中添加 `charset` 与 `viewport` 声明。

归档页面仅作为资料保存，不保证持续更新或兼容所有浏览器版本。

## 维护说明

### 本地预览

克隆本仓库后，直接在浏览器中打开任意 `.html` 文件即可预览。

```bash
git clone https://github.com/your-username/pages-cn-cns.git
cd pages-cn-cns
```
